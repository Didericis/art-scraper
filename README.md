# art-scraper
A way for getting content for artfunknet

### Requirements

* Chrome
* Selenium driver - [download](https://sites.google.com/a/chromium.org/chromedriver/downloads)

### Installation

1. Clone
* Run `npm install`

### Run

`node selenium.js`
